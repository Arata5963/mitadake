# デザイン原則

## コンセプト

**モノトーン**で統一。シンプルで洗練された印象。

## 基本原則

### 1. シンプルさ

- 不要な装飾を排除
- 白と黒を基調とした配色
- 余白を活かしたデザイン

### 2. 一貫性

- 同じ要素は同じスタイルを使用
- コンポーネントを再利用
- 命名規則を統一

### 3. アクセシビリティ

- コントラスト比 4.5:1 以上
- 最小タッチターゲット 44px
- フォーカス状態を明確に

### 4. レスポンシブ

- モバイルファースト
- ブレークポイントで適切に調整
- コンテンツの優先順位を維持

## デザインの方向性

| 要素 | 方向性 |
|------|--------|
| 色 | モノトーン（白・黒・グレー） |
| 角丸 | 控えめ（rounded-lg, rounded-xl） |
| 影 | 軽め（shadow-sm） |
| フォント | システムフォント |
| アニメーション | 控えめ（200ms） |
