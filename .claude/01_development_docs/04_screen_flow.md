# 画面遷移

## 主要画面一覧

| 画面 | パス | 説明 |
|------|------|------|
| ホーム | / | 未ログイン時のランディングページ |
| 動画一覧 | /posts | YouTube動画一覧 |
| 動画詳細 | /posts/:id | 動画詳細・アクションプラン一覧 |
| 新規投稿 | /posts/new | YouTube動画+アクションプラン作成 |
| マイページ | /mypage | 自分のアクションプラン一覧 |
| ユーザー詳細 | /users/:id | 他ユーザーのプロフィール |
| プロフィール編集 | /users/:id/edit | プロフィール編集 |

## 画面遷移図

```
┌─────────────┐
│   ホーム    │ ─── ログイン ───→ ┌─────────────┐
│     /       │                   │  動画一覧   │
└─────────────┘                   │   /posts    │
                                  └─────────────┘
                                        │
                    ┌───────────────────┼───────────────────┐
                    ▼                   ▼                   ▼
             ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
             │  新規投稿   │     │  動画詳細   │     │ マイページ  │
             │ /posts/new  │     │ /posts/:id  │     │  /mypage    │
             └─────────────┘     └─────────────┘     └─────────────┘
                    │                   │                   │
                    └───────────────────┴───────────────────┘
                                        ▼
                                 ┌─────────────┐
                                 │ 達成記録    │
                                 │ (モーダル)  │
                                 └─────────────┘
```

## 認証フロー

```
未ログイン ──→ /users/sign_in ──→ ログイン成功 ──→ /posts
                    │
                    ├──→ /users/sign_up ──→ 登録成功 ──→ /posts
                    │
                    └──→ Google OAuth ──→ 認証成功 ──→ /posts
```

## 投稿フロー

```
/posts/new
    │
    ├──→ YouTube検索 ──→ 動画選択
    │
    ├──→ YouTube URL入力 ──→ 動画情報自動取得
    │
    └──→ アクションプラン入力 ──→ 期限設定 ──→ 投稿 ──→ /posts/:id
```

## 達成フロー

```
/posts/:id
    │
    └──→ 達成ボタン ──→ 達成モーダル
                            │
                            ├──→ 感想入力
                            ├──→ 画像アップロード（任意）
                            │
                            └──→ 保存 ──→ 達成完了
```
