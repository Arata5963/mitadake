# コンセプト・要件定義

## サービス名

**mitadake?（みただけ？）**

## ビジョン

> 「見ただけで終わらせない」
>
> YouTube動画から得た学びを行動に変え、小さな一歩の積み重ねで大きな変化を生み出す

## 解決する課題

- YouTube動画で得た「これは大切だ」という気づきが日常に埋もれてしまう
- 動画視聴と実践のタイミングに時間的ギャップがある
- 学びを行動に移すための具体的な仕組みがない

## ソリューション

- YouTube URLを即座に記録できるシンプルなインターフェース
- 動画視聴直後に具体的なアクションプランを設定
- 期限付きの目標設定で実践を促進
- 達成記録の可視化でモチベーション維持

## ターゲットユーザー

**忙しい現代人で成長意欲のある層（20代〜40代）**

- 自己啓発や学習に関心があるビジネスパーソン
- 複雑な操作より「簡単で今すぐ終わる」ことを重視
- スモールステップで着実に成長したい人

## 実装済み機能

### 認証機能

| 機能 | 説明 |
|------|------|
| ユーザー登録 | メール・パスワードで登録 |
| ログイン/ログアウト | セッション管理 |
| Googleログイン | OmniAuth連携 |
| パスワードリセット | メール送信 |
| プロフィール編集 | 名前・アバター画像 |

### YouTube動画機能

| 機能 | 説明 |
|------|------|
| 動画検索 | YouTube API連携でリアルタイム検索 |
| 動画情報自動取得 | タイトル・チャンネル名・サムネイル |
| AI要約 | Gemini APIによる動画要約 |
| AIアクションプラン提案 | 動画内容からアクションプラン候補を生成 |

### アクションプラン機能

| 機能 | 説明 |
|------|------|
| アクションプラン作成 | 動画に対して行動目標を設定 |
| 期限設定 | デフォルト7日後、変更可能 |
| 達成記録 | 達成ボタンで完了を記録 |
| 感想記録 | 達成時に振り返りを記入 |
| 結果画像アップロード | 達成の証拠画像をS3に保存 |
| いいね | 他ユーザーのアクションプランに応援 |

### マイページ機能

| 機能 | 説明 |
|------|------|
| プロフィール表示 | アバター・名前・お気に入り引用 |
| アクションプラン一覧 | 自分の未達成/達成済み一覧 |
| 達成ランキング | 期間別（週間/月間/全期間） |

---

## 非機能要件

| 項目 | 要件 |
|------|------|
| テストカバレッジ | 80%以上 |
| パフォーマンス | ページ読み込み3秒以内 |
| セキュリティ | Brakeman警告ゼロ |
| レスポンシブ | モバイルファースト |

## 使用API

| API | 用途 |
|-----|------|
| YouTube Data API v3 | 動画検索・情報取得 |
| Gemini API | 動画要約・アクションプラン提案 |
| AWS S3 | 画像アップロード |
